## 🎯 Learning Objectives

By working through the projects in this repository you will be able to:

- Draw and explain a multi-component web infrastructure diagram
- Describe the role of a **DNS record** and the difference between record types
- Explain what a **web server** does vs. what an **application server** does
- Describe how a **load balancer** distributes traffic and the tradeoffs of Round-Robin vs. other algorithms
- Explain **Active-Active** vs. **Active-Passive** cluster setups
- Design a **Primary-Replica** database cluster and explain its read/write semantics
- Identify **Single Points of Failure (SPOF)** in an infrastructure and propose mitigations
- Explain why firewalls, HTTPS, and monitoring are essential for a production system
- Describe the benefits of separating components (web, app, DB) onto dedicated servers
